üìã CSV Format Instructions

The app can automatically detect common bank CSV formats. Your CSV should contain at least these columns:

  • Date: Transaction date (Date, Transaction Date, Posted Date)
  • Description: Transaction description (Description, Memo, Transaction)
  • Amount: Transaction amount (Amount, Debit, Credit)

Example CSV format:

Date,Description,Amount
2024-01-15,"GROCERY STORE #123",-85.42
2024-01-14,"PAYCHECK DEPOSIT",2500.00
2024-01-13,"GAS STATION",-45.20
2024-01-12,"AMAZON.COM",-29.99

üí° Tips:

  • Negative amounts represent expenses, positive amounts represent income
  • The app will automatically categorize transactions based on description
  • You can upload multiple files - they'll all be combined
  • Duplicate transactions are handled automatically
  • Supported date formats: YYYY-MM-DD, MM/DD/YYYY, DD/MM/YYYY
